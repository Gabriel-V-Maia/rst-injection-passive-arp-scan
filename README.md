# rst-injection-passive-arp-scan

Tentativa de enumerar dispostivios com um scan passivo dando sniff em pacotes ARP e fechamentos de conexões TCP via RST injection

O bom de usar o ARP como método para enumerar dispositivos, é que ele não causa ruido algum na rede, em contrapartida caso a rede for bem protegida pode não funcionar :(
